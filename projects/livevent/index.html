livEvent 🎵

Aplicación para gestión de eventos y red social para promoción de artistas

Source code livEvent

Created Dec 15, 2019 - Last updated: Dec 15, 2019

Evergreen 🌳
php api

livEvent es la aplicación que gestiona eventos de todo tipo, promocionándolos y permitiendo una mejora en la interacción seguidor-artista. La red social definitiva para artistas locales que buscan promocionar sus eventos y para seguidores que no quieren perderse ninguna de sus actuaciones.

  • Sigue a tus artistas favoritos y los eventos que te interesen.
  • Descubre artistas locales, sus próximas actuaciones, dónde comprar entradas…
  • Encuentra nuevos eventos cerca de tí, se acabaron los fines de semana sin planes.
  • Muestra tu apoyo valorando los eventos con una sencilla encuesta dentro de la app.
  • ¿Eres artista? Promocionate creando un perfil en livEvent y aumenta tu audiencia.
